# koa-passport-jwt-api

Using Koa for REST APIs with Passport and JWT as an authentication token without sessions.

## Uses

1.  "koa": "^2.4.1",
2.  "koa-passport": "^4.0.1",
3.  "koa-router": "^7.3.0",
4.  "passport-jwt": "^4.0.0",

## Getting Started

1. Fork/Clone
1. Install dependencies - `npm install`
1. Run - `npm start`
